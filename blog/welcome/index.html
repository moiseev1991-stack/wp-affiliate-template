Welcome to your new site

This is a placeholder post that ships with the template so that the static export build succeeds.

Run the /scaffold-site skill in Claude Code, or trigger the weekly post action manually, and this file will be replaced with a real article in your chosen language and niche.

What happens next

  • The scaffold-site skill rewrites all UI strings, configures your domain, creates a GitHub repo, and sets the OPENAI_API_KEY secret.
  • The weekly GitHub Action picks a fresh topic each Monday and writes a 1600+ word article.

Customizing further

Edit lib/config.ts, components/Header.tsx, components/Footer.tsx, and scripts/generate-post.mjs to fine-tune branding and topic categories.
